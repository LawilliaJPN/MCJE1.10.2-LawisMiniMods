# MCJE1.10.2-LawisMiniMods  
  
Lawi's Mini Mods(Minecraft Java Edition ver.1.10.2)  
  
## 詳細(Details)  
  
### 概要(Overview)  
マイクラJava版ver.1.10.2のための、  
製作した小規模単発modのコードを置いておくための場所です。  
  
### 動作環境(Requirements)  　
* Minecraft Java Edition ver.1.10.2  
* Minecraft Forge ver.1.10.2-12.18.2.2099
